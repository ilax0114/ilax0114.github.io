1. HTML en CSS werken samen om webpagina's te structureren en te stylen

HTML (HyperText Markup Language) vormt de basis van een webpagina en beschrijft de inhoud en structuur, zoals koppen, paragrafen en afbeeldingen. CSS (Cascading Style Sheets) wordt gebruikt om de presentatie van die inhoud te verbeteren door kleuren, lettertypen, marges en andere visuele stijlen toe te voegen. Samen vormen ze de kern van webontwikkeling.

CSS maakt gebruik van het 'Cascading'-principe om stijlen toe te passen.

Het 'Cascading'-principe betekent dat de browser bepaalt welke CSS-regels worden toegepast wanneer meerdere regels op hetzelfde element van toepassing zijn. Dit gebeurt door prioriteit te geven aan inline styles, gevolgd door stijlen in een style blok, en ten slotte stijlen in externe stylesheets. Ook specifieke selectors hebben meer prioriteit dan algemene.

Responsive design wordt mogelijk gemaakt door CSS media queries.

Met media queries in CSS kun je stijlen aanpassen op basis van schermgrootte, resolutie of apparaattype. Dit is essentieel voor het maken van websites die goed werken op zowel desktops als mobiele apparaten.
